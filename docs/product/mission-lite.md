# Product Mission (Lite)

spec-watcher is a full-screen terminal dashboard that helps developers keep a live eye on a repository's specs by continuously watching the specs folder and displaying every spec's name, description, status, and task progress in one sleek, always-current table.

spec-watcher serves developers and technical leads who track work through the Agent OS `docs/specs` convention and need an at-a-glance progress board. Unlike a generic markdown viewer or a heavyweight project tool, it understands the exact `spec.md` / `spec-lite.md` / `tasks.md` layout, derives real progress from checkbox completion alongside the declared status, and watches on a non-blocking timer with on-demand refresh — all from a single zero-setup CLI.
